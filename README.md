# POLICITY

##### POLICITY은 청년층의 정치 참여 장벽을 낮추기 위해 설계된 AI 기반 정치 입문 플랫폼이다.
서비스는 두 가지 핵심 기능으로 구성된다.
청년정책을 중심으로 주요 이슈와 관심 키워드 정보를 일상 대화체로 전달하는 뉴스·정책 정보 챗봇
찬반 입장을 선택하여 AI와 1대1로 토론하거나 AI 간 토론을 참관하는 AI 토론 기능
챗봇은 사용자의 관심사를 파악하고 이 두 기능을 자연스럽게 연결하는 허브 역할을 수행한다. 이 서비스의 핵심 목표는 사용자가 정치 지식을 '공부'가 아니라 자신의 삶과 연결된 경험으로 받아들이도록 돕는 것이다. 특정 정당이나 후보를 지지·반대하는 결론을 제시하지 않고, 다양한 관점을 균형 있게 제공하는 편향 방지 설계를 원칙으로 한다.
